About This System

What is Fetal Image Analysis?

This AI-powered system uses YOLOv8 deep learning to automatically classify fetal ultrasound images into 9 anatomical planes. It assists prenatal screening by identifying structures like the fetal brain, abdomen, femur, thorax, and more.

Trained on the BCNatal Maternal-Fetal Ultrasound Dataset (12,400+ images from Zenodo), the model achieves high accuracy in distinguishing between different anatomical views, helping clinicians quickly verify standard diagnostic planes.
